Instalación Visual Studio

Introdución.

El siguiente documento tiene como principal objetivo, servir de guía al estudiantado para facilitarle la instalación del software Visual Studio IDE, que se recomienda utilizar a lo largo del curso Programación Orientada a Objetos, ciclo 01 2019.

Descargar Visual Studio

Para iniciar se debe abrir el navegador web de preferencia, estando dentro, copiar y pegar la siguiente dirección https://visualstudio.microsoft.com/es/thank-you-downloading-visual-studio/?sku=Community&rel=15, puedes acceder a la pagina presionando la tecla Ctrl + clic en el vínculo y se abrirá automáticamente en el navegador por defecto, al finalizar este proceso se mostrara la siguiente pagina web:

A continuación se debe iniciar automáticamente una descarga que contiene el instalador de Visual Studio, si no ocurre automaticamente, hacer clic en el link "Haga clic aquí para volver a intenarlo."

Instalar Visual Studio

Para iniciar la instalación de Visual Studio, se debe abrir el instalador previamente descargado, se mostrara una ventana en la cual se pueden consultar los terminos y condiciones del software, para iniciar la instalación hacer clic en "continue"

Se iniciará un proceso de descarga total del software seguido de su instalación.

Al finalizar ambos procesos, seleccionar el ".NET desktop development" y hacer clic en "install", seguidamente de esta acción se iniciara la descarga del componente previamente seleccionado y al finalizar la descarga, se iniciara la instalación

Una vez realizada esta acción, se iniciara el proceso de descarga e instalación, al finalizar ambos procesos la instalación se habra realizado con éxito
