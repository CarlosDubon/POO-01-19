Programación Estructurada

Introdución.

El siguiente documento tiene como objetivo principal retroalimentar las sentencias y estructuras básicas, es importante saber el funcionamiento de cada una de ellas, ya que, en la programación orientada a objetos, estas sentencias siguen utilizándose en gran medida.

Sentencias condicionales

Se definen como instrucciones o grupo de instrucciones que se pueden ejecutar o no en función a un valor de condición.

Sentencia IF

Ejecuta un bloque de instrucciones si la condición es verdadera, además pueden omitirse las llaves de apertura y de cierre si solo se va a ejecutar una instrucción. 

                    
                        if (condición)
                            // si solo se ejecutara una instrucción se pueden omitir las llaves { }

                        if (condición) {
                            // bloque de código a ejecutarse si la condición es verdadera
                        }

Ejemplo 

                    
                        if (50 > 28)
                            System.out.println("50 es mayor que 28");

Sentencia IF-ELSE

Ejecuta un bloque de instrucciones si la condición es falsa, sin embargo esta palabra reservada else es opcional, ya que es necesaria solo si se necesita ejecutar un conjunto de instrucciones si la condición es falta. Por lo tanto si la condición del if no se cumple y no queremos realizar una acción al no cumplirse la condición, podemos usar solamente la sentencia if.

Sintaxis 

                    
                        if (condición) {
                            // bloque de código a ejecutarse si la condición es verdadera
                        } else {
                            // bloque de código a ejecutarse si la condición es falsa
                        }

Ejemplo 

                    
                        if (nota_Final_POO>= 6) {
                            System.out.println("La pase! :D");
                        } else {
                            System.out.println(" :( ");
                        }

Operedor Ternario (IF de una linea);

Otra forma de sintaxis para el uso del if-else, y resumirlo en pocas líneas de código, además al igual que en el if-else podemos anidar sentencias condicionales.

Sintaxis 

                    
                        (condición)?expresionVerdadera:expresionFalsa;
                        /*anidamiento de operador ternario*/
                        (condicion1) ? (ExpresionVerdadera: condicion2) ? ExpresionVerdadera:ExpresionFalsa;

Ejemplo 

                    
                        Calendar cal = Calendar.getInstance();
                        int hora= cal.get(Calendar.HOUR_OF_DAY);
                        String resultado = (hora < 7) ? "puedo seguir durmiendo" : "Es hora de ir a la universidad";
                        System.out.println(resultado);

IF-ELSE Anidados

Podemos volver a usar sentencias condicionales if luego de un else, es lo que se conoce como if-else anidados, ya que a veces se presenta el caso que necesitamos evaluar más condiciones o expresiones por si la expresión o condición inicial no se ha cumplido y a partir de ello ejecutar una secuencia de instrucciones.

Sintaxis 

                    
                            if (condición) {
                                // bloque de código a ejecutarse si la condición es verdadera
                            } else {
                                /*Inicio de anidamiento*/
                                if (condición) {
                                    // bloque de código a ejecutarse si la condición es verdadera
                                }
                                else {
                                    if (condicion){
                                        // Bloque de código
                                    } else {
                                        /* Podríamos continuar el anidamiento más veces*/
                                    }
                                }
                            }

Ejemplo 

                    
                            int valor=5
                            //tomando en cuenta números enteros positivos
                            if (valor>= 0 && valor<=10) {
                                System.out.println("El valor esta en el rango de 0 a 10");
                            } else {
                                if (valor>10 && valor<=15) {
                                    System.out.println("El valor está en el rango de 11 a 15");
                                } else{
                                    System.out.println("El valor es mayor a 15");
                                }
                            }

Sentencia SWITCH

Ejecuta un bloque de instrucciones dependiendo del valor de la expresión evaluada

Sintaxis 

                    
                            switch(expresión) {
                                case a:
                                // bloque de código
                                    break;
                                case b:
                                // bloque de código
                                    break;
                                default:
                                // bloque de código
                            }

Ejemplo 

                    
                            int day = 4; switch (day) {
                                case 1:
                                System.out.println("Lunes");
                                    break;
                                case 2:
                                    System.out.println("Martes");
                                    break;
                                case 3:
                                    System.out.println("Miercoles");
                                    break;
                                case 4:
                                    System.out.println("Jueves");
                                    break;
                                case 5:
                                    System.out.println("Viernes");
                                    break;
                                case 6:
                                    System.out.println("Sabado");
                                    break;
                                case 7:
                                    System.out.println("Domingo");
                                    break;
                                default:
                                    System.out.println("el valor no corresponde al rango de días de la semana");
                            }

Sentencias iterativas

Dentro de la programación, los bucles juegan un papel fundamental para la resolución de problemas en donde necesitamos realizar un número determinado de veces una tarea o por lo menos cuantas veces sea necesario. Los bucles también son conocidos como ciclos y son estructuras de control que permiten modificar el flujo de ejecución de las instrucciones de un programa por medio de iteraciones. Los bucles se basan sobre este concepto:

Existen diferentes tipos de ciclos y cada uno de ellos tiene su manera de ser implementado dependiendo del contexto del problema planteado. En Java encontramos los siguientes: ciclo For, ciclo While y ciclo Do-While.

Ciclo FOR

El objetivo de este ciclo es realizar un número determinado de veces una tarea. Dentro de la estructura del ciclo for se define una variable de control que en la mayoría de casos actúa como acumulador, este se encarga de contabilizar el número de veces que se ejecutará una instrucción. Posteriormente se establece una condición de paro en donde nuestra variable de control que actúa como acumulador debe ir implicada ya que es donde establecemos la condición de paro respecto al número de veces que deseamos ejecutar una instrucción. Por último, tenemos que hacer una operación matemática con nuestro acumulador para poder contabilizar el número de veces que se ejecutará la instrucción.

Sintaxis 

                    
                        for([acomulador];[condition];[operación]){
                            //Bloque de codigo
                        }

Ejemplo 

                    
                        for(int i=0;i < 10; i++){
                            System.out.println(i+" x 5 =" + (5*i))
                        }

Ciclo WHILE

Este ciclo es basado en los resultados de una expresión lógica o condicional. El propósito es repetir un bloque de código mientras una condición se mantenga verdadera.

Sintaxis

                    
                        while([condition]){
                            //Code
                        }

Ejemplo 

                    
                        while(true){
                            //Bucle infinito
                        }

Cliclo DO-WHILE

Al igual que el ciclo while, este bucle se basa en la operación de una expresión lógica. El Do-while se diferencia en la manera en la que ejecuta el flujo de datos, ya que primero ejecuta el bloque de instrucciones y posteriormente hace la operación lógica de comparación. Con lo cual se garantiza que se ejecute siempre por lo menos una vez

Sintaxis 

                    
                        do{
                            //code
                        }while([condition])

Estructura de datos

Nos proveen una forma de administrar la información, de tal forma que se pueda manipular, realizar operaciones de busqueda e insertar datos de manera eficiente. Ejemplos de estructuras de datos son las siguientes:

  • Arrays (Arreglos)
  • Lists (Listas enlazadas)
  • Collections (Colecciones)

 
Arrays (Arreglos)

Los arreglos son una estructura de datos que, una vez se inicializa con un tamaño específico, ya no puede cambiar. Importante destacar que solo admite almacenar datos de un mismo tipo, una vez declarada. La forma en la que se declara un arreglo es la siguiente: 

tipo_de_dato[] arreglo;
Así se declara un arreglo, pero es necesario especificar un tamaño para poder comenzar a usarlo. Para ello se hace lo siguiente: 
arreglo = new tipo_de_dato[tamaño_de_arreglo];
Donde tipo_de_dato puede ser: int, float, double, boolean, cualquier instancia de objeto, etc. El indice en el que comienza a contar, como en la mayoría de los lenguajes de programación, es cero. Si se recorre el arreglo sin haber llenado sus posiciones con un valor; el valor por defecto es cero para cada posición. Ejemplo:

  package arreglolistas;

  import java.util.ArrayList;
  import java.util.List;
  import java.util.Scanner;


  public class ArregloListas {

      /**
       * @param args the command line arguments
       */
      public static void main(String[] args) {
          Scanner entrada = new Scanner(System.in);
          int[] arreglo = new int[10];
          boolean flag = true;


          for (int i = 0; i < arreglo.length; i++) {
              System.out.print("Ingrese un numero al arreglo: ");
              arreglo[i] = entrada.nextInt(); entrada.nextLine();
          }

          do{
              System.out.println("***¿Que desea hacer?***");
              System.out.println("1-Llenar arreglo\n2-Mostrar arreglo\n3-Encontrar mayor\n4-Encontrar menor\n5-Encontrar promedio\n6-Salir");
              int decision = entrada.nextInt(); entrada.nextLine();

              switch(decision){
                  case 1:
                      llenarArreglo(arreglo, entrada);
                      mostrarArreglo(arreglo);
                      break;
                  case 2:
                      mostrarArreglo(arreglo);
                      break;
                  case 3:
                      mostrarArreglo(arreglo);
                      System.out.println("El numero mayor es: " + encontrarMayor(arreglo));
                      break;
                  case 4:
                      mostrarArreglo(arreglo);
                      System.out.println("El numero mayor es: " + encontrarMenor(arreglo));
                      break;
                  case 5:
                      mostrarArreglo(arreglo);
                      System.out.println("El numero mayor es: " + encontrarPromedio(arreglo));
                      break;
                  case 6:
                      flag = false;
                      break;
              }

          }while(flag);


      }


      public static void llenarArreglo(int[] arreglo, Scanner entrada){
          for (int i = 0; i < arreglo.length; i++) {
              System.out.println("Ingrese valor:");
              arreglo[i] = entrada.nextInt();
          }
      }

      public static void mostrarArreglo(int[] arreglo){
          for (int i = 0; i < arreglo.length; i++) {
              System.out.print("[" + arreglo[i] + "]");
          }
          System.out.println();
      }

      public static int encontrarMayor(int[] arreglo){

             int mayor = arreglo[0];

          for (int i = 0; i < arreglo.length; i++) {

              if(arreglo[i] > mayor){
                  mayor = arreglo[i];
              }
          }
          return mayor;
      }


      public static int encontrarMenor(int[] arreglo){

          int menor = arreglo[0];

          for (int i = 0; i < arreglo.length; i++) {

              if(arreglo[i] < menor){
                  menor = arreglo[i];
              }
          }
          return menor;
      }

      public static double encontrarPromedio(int[] arreglo){

          double suma = 0;

          for (int i = 0; i < arreglo.length; i++) {
              suma += arreglo[i];
          }
          return suma / arreglo.length;
      }


  }


                
              

 
Collections (ArrayList)

Es una colección de elementos. Lo importante del termino ‘colección’ es que se caracterizan por ser dinámicos, a diferencia de los arreglos, los cuales necesitan que se especifique el tamaño sobre el que se trabajará. De igual forma, una característica importante es que los elementos que se almacenan en una colección pueden ser de cualquier tipo de dato, ya que los valores que se guardan se consideran elementos de tipo ‘object’, lo que significa que pueden ser: Integer, Float, String,Boolean, una instancia de clase, etc. Para utilizarlo es necesario declararlo de la siguiente manera: 

ArrayList lista = new ArrayList<>();
De esta manera se tiene un arreglo dinámico de objetos (pueden ser de cualquier tipo.). Sin embargo a veces se quiere especificar el tipo de dato que aceptará únicamente. Para ello se escribe de la siguiente manera: 
ArrayList<tipo_de_dato> lista = new ArrayList<tipo_de_dato>();
Estas listas tiene funciones asociadas, por el momento diremos que son métodos de la lista. Dentro de estas, encontramos las siguientes:
  • add(Object); Para añadir elementos a la lista. Se ingresan en secuencia, se le asigna un indice a cada elemento, y dentro de las listas pueden existir valores repetidos.
  • get(int); Para obetener elementos de la lista respecto a indice. Comienza con indice 0
  • size(); Retorna el tamaño del ArrayList
  • contains(Object); Para buscar elementos dentro de la lista. Si lo encuentra regresa true, de otra forma regresa false
  • remove(Object); Para eliminar un elemento dentro de la lista. Si lo encuentra regresa true, de otra forma regresa false
Para utilizar estos métodos se debe tomar la lista que se declaro más arriba (en la que se acepta cualquier tipo de objeto) y considerar la siguiente notación: 
                  
  lista.add("Este es un texto");

  lista.add(2);

  lista.add(true);

Funciones

Se pueden categorizar funciones el conjunto de lineas de instrucciones, encapsulados en un bloque, usualmente reciben parámetros, cuyos valores se utilizan para efectuar operaciones y adicionalmente retornan un valor, en caso de no retornar un valor, no se categorizan como una función. En java las funciones usan el modificador static.

Sintaxis 

                
                    public static [tipo_dato_a_retornar|void] [nombre_función] ([tipo_dato][parametro1],[tipo_dato][parametro2]...){
                        //Bloque de código aquí
                    }

Ejemplo 

                
                    public static void main(){
                        int[] numbers = {1,2,3,4};
                        int c = sumaEnteros(numbers); //<- se obitene el resultado en la variable c
                    }
                    public static int sumaEnteros(int[] a){
                        int b = 0;
                        for(int i=0 ; i < a.length ; i++){
                            b = b + a[i] ;
                        }
                        return b;
                    }
